# Arena Coach - Frontend (Next.js + Tailwind)

Frontend pronto para conectar ao backend AirCoach (endpoints já gerados). Visual: esportivo e energético (tons de areia e laranja). Pronto para deploy no Vercel.

## Como usar

1. Unzip the project.
2. Copy `.env.local.example` to `.env.local` and fill variables.
3. `npm install`
4. `npm run dev`

Deploy recomendado: Vercel. Configure env vars in Vercel same as `.env.local`.
